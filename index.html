HTML, CSS, JS SELF ASSESSMENT

This is the main section. Below are the list of top 10 html tags i feel are most important.

  • Div - helps with creating sections in the page. makes it easier to style.
  • Span - helpful for grouping inline content.
  • a - hyperlink tag.
  • Table tag
  • ul & li: used the most i feel, even in nav bars,footers
  • button: used in forms, going to the next page, searching saving so many things
  • Script: we will need this tag to wrtie js code for any action our buttons, links, etc. will do<
  • form: login, registrations, contact info, feedback, all need this tag
  • header, footer: essential for the website owner, organization to summarize, add links, copyright infor etc.
  • datalist: easier to search, better than select, makes it easier to adapt to diff data types.
